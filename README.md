# Bateria Virtual

Bem-vindo à aplicação Bateria Virtual! Esta é uma aplicação web que simula uma bateria, permitindo que os usuários toquem diferentes partes do instrumento como em uma bateria real. A aplicação foi construída utilizando JavaScript, HTML e CSS.

## Instruções de Uso

1. Abra o seu navegador web moderno, como Google Chrome, Mozilla Firefox ou Microsoft Edge.

2. Clone este repositório para o seu computador ou faça o download como arquivo ZIP.

3. Navegue até o diretório onde você clonou ou descompactou o repositório.

4. Abra o arquivo `index.html` em seu navegador.

5. Na interface da aplicação, você verá diferentes partes da bateria, como caixa, bumbo, prato, entre outros.

6. Clique nas diferentes partes da bateria usando o mouse ou toque na tela, e você ouvirá o som correspondente da bateria.

7. Divirta-se tocando diferentes combinações e ritmos!

## Tecnologias Utilizadas

- JavaScript: Linguagem de programação usada para criar a lógica por trás da aplicação e controlar a reprodução dos sons da bateria.

- HTML: Linguagem de marcação usada para estruturar a interface da aplicação.

- CSS: Linguagem de estilo usada para estilizar a aparência dos elementos HTML e criar uma interface agradável.

## Funcionalidades da Aplicação

- **Interatividade:** Cada parte da bateria é clicável e produz um som correspondente ao ser clicado.

- **Reprodução de Som:** A aplicação utiliza arquivos de áudio para reproduzir os sons realistas de uma bateria quando as partes são acionadas.

- **Variedade de Sons:** Diferentes partes da bateria têm sons distintos, permitindo que o usuário crie ritmos variados.
